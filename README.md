# Architecture and Urban Planning Zoning Research Chrome Extension

A Chrome extension for architects, urban planners, and real estate developers browsing municipal GIS portals and zoning databases. Extracts zoning classifications, permitted use tables, height limits, setback requirements, FAR calculations, and parking minimums directly from zoning code pages and displays them in a structured sidebar. Tracks researched parcels in a project workspace, exports zoning summaries to PDF, and flags when proposed uses require a conditional use permit or variance.

## Installation

### Chrome
1. Download and extract the ZIP
2. Go to `chrome://extensions/`
3. Enable "Developer mode"
4. Click "Load unpacked" and select the extracted folder

### Firefox
1. Download the Firefox version
2. Go to `about:debugging`
3. Click "Load Temporary Add-on"
